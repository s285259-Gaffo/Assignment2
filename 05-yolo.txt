Problem formulation
n Given an image generate bounding 
boxes, one for each detectable object in 
image
n For each bounding box, output 5 
predictions: x, y, w, h, confidence. Also 
output class
n x, y (coordinates for center of bounding box)
n w,h (width and height)
n confidence (probability bounding box has 
object)
n class (classification of object in bounding 
box)

YOLO algorithm
n A single neural network for localization 
and for classification (less complicated 
pipeline)
n Need to inference only once (efficient 
computation)
n Looks at the entire image each time 
leading to less false positives (has 
contextual information for detection)
n YOLO operations:
n Image is split into a SxS grid
n For each grid square, generate B bounding 
boxes
n For each bounding box, provide 5 predictions: 
x, y, w, h, confidence
